# Linux + Git Cheat Sheet (15 commands)

> Format: **Command** — what it does — example

## Linux Commands
1. **pwd** — print current directory — `pwd` (useful before running cd)
2. **ls** — list files — `ls -la`  (shows hidden files with -a)
3. **cd** — change directory — `cd ..`
4. **mkdir** — create a folder — `mkdir projects`
5. **touch** — create empty file — `touch notes.txt`
6. **cp** — copy files — `cp a.txt b.txt`
7. **mv** — move/rename — `mv old.txt new.txt`
8. **rm** — delete — `rm file.txt`
9. **cat** — view file content — `cat readme.md`
10. **grep** — search text — `grep "main" file.txt`

## Git Commands
11. **git status** — show changes — `git status`
12. **git add** — stage changes — `git add .`
13. **git commit** — save snapshot — `git commit -m "message"`
14. **git branch** — list/create branches — `git branch`
15. **git merge** — merge branch into current — `git merge branch-name`
